Приветствую тебя путник....

Первое что ты должен сделать салить све пакеты gulpa

вот список необходимых тебе пакетов а остальное уже настроено за тебя

1. Не забудь убедится что у тебя есть node.js
2. Установи gulp глобально вот команда
npm install --global gulp-cli
Если ты раньше устанавливал на компьютер gulp то тебе нужна команда
npm rm --global gulp

3. Следующая команда для создания json файла. Лови команду...
npm init

Сокращенная форма для автоматического заполнения проекта

npm init -y

4. Дальше тебе нужно установить gulp именно для своего проекта вот команда
npm install gulp --save-dev

5. установка локального сервера
npm i browser-sync --save-device

6. Установка сборщика всех файлов в один
npm i gulp-file-include --save-dev

7. Установка пакета удаления ненужных файлов
npm i del --save-dev

8. Установка SCSS преобразователя
npm i gulp-sass --save-dev

9.Тебе также понадобится avtoprefixer
npm i gulp-autoprefixer --save-dev

10. Преднозначен для сборки медиа запросов в конец стилей
npm i gulp-group-css-media-queries

11. Чистит и сжимает css
npm i gulp-clean-css

12. Переименовывает файла
npm i gulp-rename --save-dev

13.Сжатие js
npm i --save-dev gulp-uglify-es

14.Сжатие картинок
npm i gulp-imagemin

15. Установка плагина WebP
npm i gulp-webp --save-dev

16.Плагин для отображения picture
npm i gulp-webp-html
Удачи путник....


Если у тебя уже это все есть то просто запусти
npm i
